# Branding Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Show the KrillNotes logo on the empty-state screen and replace the placeholder Tauri icon with the KrillNotes app icon everywhere (Dock, taskbar, window decorations).

**Architecture:** Two independent changes. (1) Copy `KrillNotesLogo.png` into `public/` so Vite serves it as a static asset, then update `EmptyState.tsx` to render it. (2) Run `tauri icon` CLI to regenerate all icon sizes from `KrillNotesAppIcon.png` (the no-text variant, correct for an OS icon).

**Tech Stack:** React 18, TypeScript, Tauri v2, Vite. No frontend test suite — verification is manual (`npm run tauri dev` in `krillnotes-desktop/`).

---

### Task 1: Add logo to empty state

**Files:**
- Copy: `marketing/KrillNotesLogo.png` → `krillnotes-desktop/public/KrillNotesLogo.png`
- Modify: `krillnotes-desktop/src/components/EmptyState.tsx`

**Step 1: Copy the logo into the Vite public folder**

```bash
cp marketing/KrillNotesLogo.png krillnotes-desktop/public/KrillNotesLogo.png
```

Vite serves everything in `public/` at the root URL, so the image will be reachable as `/KrillNotesLogo.png` at runtime.

**Step 2: Replace `EmptyState.tsx` with this content**

```tsx
function EmptyState() {
  return (
    <div className="flex items-center justify-center min-h-screen">
      <div className="text-center">
        <img
          src="/KrillNotesLogo.png"
          alt="KrillNotes"
          className="w-64 h-64 mx-auto mb-6 object-contain"
        />
        <p className="text-muted-foreground">
          Use File &gt; New Workspace or File &gt; Open Workspace to get started
        </p>
      </div>
    </div>
  );
}

export default EmptyState;
```

Key points:
- `w-64 h-64` = 256 × 256 px display size — large enough to read but not overwhelming
- `object-contain` preserves aspect ratio without cropping
- The `<h1>` text is removed because the logo already includes the "KrillNotes" wordmark

**Step 3: Verify TypeScript compiles**

```bash
cd krillnotes-desktop && npx tsc --noEmit
```

Expected: no errors.

**Step 4: Commit**

```bash
git add krillnotes-desktop/public/KrillNotesLogo.png krillnotes-desktop/src/components/EmptyState.tsx
git commit -m "feat(frontend): show KrillNotes logo on empty state screen"
```

### Task 2: Replace app icon with KrillNotes icon

**Files:**
- Regenerate: `krillnotes-desktop/src-tauri/icons/` (all files)

**Step 1: Run the Tauri icon generator**

From the repo root:

```bash
cd krillnotes-desktop && npm run tauri icon -- ../marketing/KrillNotesAppIcon.png
```

`tauri icon` reads the 1024×1024 source PNG and writes all required variants into `src-tauri/icons/`:
- `32x32.png`
- `128x128.png`
- `128x128@2x.png` (256×256)
- `icon.icns` (macOS multi-resolution bundle)
- `icon.ico` (Windows multi-resolution bundle)
- Various Windows Store `Square*.png` sizes

The `tauri.conf.json` already references the correct paths, so no config change is needed.

**Step 2: Verify the icons were generated**

```bash
ls -lh src-tauri/icons/
```

Expected: `icon.icns`, `icon.ico`, `32x32.png`, `128x128.png`, `128x128@2x.png` all present and non-zero size. The `.icns` and `.ico` files should be noticeably larger than before (the old ones were the Tauri placeholder).

**Step 3: Commit**

```bash
git add src-tauri/icons/
git commit -m "feat(desktop): replace placeholder Tauri icon with KrillNotes app icon"
```
